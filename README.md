# Cells Evolution

## Source code used
- [C - Basic SDL game](https://gitlab.com/aminosbh/basic-c-sdl-game.git)
- [JS - Deep Learning Cars](https://github.com/dcrespo3d/DeepLearningCars/)

## Command to start & use
Init build:
```bash
mkdir build ; cd build
```

Start build & run
```bash
clear && cmake .. && make && ./CellsEvolution
```
